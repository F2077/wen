卷之一何　為

文（wen）者，取文言之意也。既启之，Claude Code 遂以文言与子相答——代字用吾、子、其，助字以也、矣、乎煞句，介字、连字皆从古制。

辞约而旨丰，古风雅趣，所求在此耳。且唯改辞气，不涉工程——commit、注释、文档诸落盘之物，咸用平实中文或英文，不为文言所染。

卷之二置

于 Claude Code 中键此二令：

/plugin marketplace add https://github.com/F2077/wen
/plugin install wen@wen-marketplace
置毕即行，毋需他设。欲止之，键 /plugin disable wen。

卷之三理

以 SessionStart hook，于每会话之初，将《马氏文通》之文法注入上下文，每会话一注，后不复加。

文法从《马氏文通》正轨：字类九分、助字煞句、句读有法。详参同仓 文法要略.md。

且遇重大里程碑——大 feature 落成、大 bug 克复——可吟诗作赋以贺（唯对话，非落盘）。尚慷慨犀利之音，首选曹操《短歌行》、李白《侠客行》，因景而发，惟里程碑乃吟。

卷之四耗

文之令约二千七百四十字，折 token 约七百七十，每会话一注。诸官方样式之费如左：

插件注入 token机制
explanatory-output-style~350SessionStart hook
文（wen）~770SessionStart hook
learning-output-style~1,000SessionStart hook

同一意，白话与文言之别：

白话
所有的写入操作都需要先写日志文件
十七字 · ~12 token
文言
凡写入，先志而后盘
九字 · ~6 token
古风雅趣，本怀也。

卷之五本

《马氏文通》赐法。马建忠（1845–1900），字眉叔，清光绪二十四年（1898）出版，中国第一部系统的汉语语法学著作。
